# TKSI — Transport Knowledge System Indonesia

Portal pengetahuan transportasi Indonesia yang menghubungkan teori, kebijakan, perencanaan, operasi, dan pengalaman pelanggan.

## Struktur awal

- `D01 Foundation of Transportation`
- `KO-001 Transportasi`
- Seri pembelajaran JUTPI
- Glosarium transportasi

Situs dirancang sebagai HTML/CSS/JavaScript statis agar dapat dijalankan langsung melalui GitHub Pages tanpa proses build.

## Publikasi GitHub Pages

Pada repository, buka **Settings → Pages**, pilih **Deploy from a branch**, lalu pilih branch **main** dan folder **/(root)**.

Alamat situs: `https://rofikk.github.io/tksi/`
